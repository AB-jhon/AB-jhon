## Principios de desarrollo

**Accesibilidad** — Implementación basada en pautas WCAG 2.1. El diseño inclusivo no es opcional, es el estándar.

**Escalabilidad** — Arquitectura de componentes desacoplados que permite crecer sin deuda técnica.

**Experiencia de usuario** — Navegación fluida, modo oscuro dinámico, diseño responsivo mobile-first.

**Entrega continua** — Integración de flujos CI/CD y gestión ágil con tableros Kanban.
